## 🌟 Overview

Welcome to my personal portfolio! I am a B.Tech student with a dual passion for writing clean, efficient code and crafting compelling visual stories. This portfolio was built from the ground up to showcase not just my projects, but my attention to detail in UI/UX design, interactive elements, and frontend performance.

## ✨ Key Features

- **Premium User Experience**: Lightning-fast navigation with a modern dark-mode aesthetic and glassmorphism panels.
- **Custom Cursor Engine**: A fully custom-built cursor system featuring multiple distinct modes (Comet Trail, Spotlight, X-Ray Lens, Orbital Scanner) that reacts dynamically to hover states.
- **Interactive Bento Grid**: A beautifully structured "About Me" section that uses asymmetrical grid layouts and layered shadows to present information.
- **Physics Playground (Easter Egg)**: An integrated 2D physics engine (Matter.js) that allows users to interact with falling elements directly in the browser.
- **Smooth Animations**: Powered by Framer Motion, featuring staggered reveals, spring physics scaling, and dynamic text glitch effects.

---

## 💻 Tech Stack

**Frontend Architecture**
- **Framework**: React.js (Vite)
- **Styling**: Vanilla CSS with custom Glassmorphism UI & CSS Variables
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Physics Engine**: Matter.js
- **Deployment**: Vercel Global Edge Network

## 🚀 Getting Started

To run this project locally on your machine, follow these steps:

### Prerequisites
- Node.js (v18+)
- npm or yarn

### 1. Clone the Repository
```bash
git clone https://github.com/Soham-0207/Portfolio.git
cd Portfolio
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Development Server
```bash
npm run dev
```
Open your browser and navigate to `http://localhost:5173` to view the live preview.
